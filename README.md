# What you need to do to get this project up-and-running?

- Add spesific cmsis lib as submodule e.g. [github.com](https://github.com/STMicroelectronics/cmsis-device-f4)

- Add spesific hal lib as submodule e.g. [github.com](https://github.com/STMicroelectronics/stm32f4xx-hal-driver)

- if necessary add further drivers

- add target spesific vscode and cmake submodules

- add core submodule
